# cat

> Print and concatenate files to stdout.
> This command is an alias of `gzip --stdout --decompress`.
> More information: <https://docs.brew.sh/Manpage#search--s-options-textregex->.

- Print the contents of a file to stdout:

`cat {{path/to/file}}`

- Redirect stdin to a file:

`cat > {{path/to/file}}`

- Write stderr to a file:

`command 2> {{path/to/file}}`

- Decompress a file and write to stdout (implies `--keep`):

`xz {{[-d|--decompress]}} {{[-c|--stdout]}} {{path/to/file.xz}}`

- Test if a given string conforms the specified regex:

`[[ ${{variable}} =~ {{pattern}} ]]`

- Test if a given string conforms the specified RegEx:

`[[ ${{variable}} =~ {{pattern}} ]]`
